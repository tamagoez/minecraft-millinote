# /execute
Execute command as player or entity.

## For example...
Use `/fill` command with absolute path as each player.

## Command Syntax
### Java Edition
```
execute ...
```

### Bedrock Edition
```
execute [target selector] [position] [command]
```
You can use **absolute** position.
