# /execute
プレイヤーやエンティティとしてコマンドを実行させることができます。

## 使用用途のイメージ
特定のプレイヤーまたはエンティティの相対座標を使用してコマンドを実行したいときなど。

## コマンドの構文
**Java版とBE版でコマンドの構文が全く異なります**

### Bedrock Edition
```
execute [ターゲットセレクター] [座標] [実行コマンド]
```

座標は相対座標でも可能です。

### Java Edition
```
execute ...
```
