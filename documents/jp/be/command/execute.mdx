# /execute
プレイヤーやエンティティとしてコマンドを実行させることができます。
<NoCompatibility />

## 使用用途のイメージ
特定のプレイヤーまたはエンティティの周辺に`/fill`コマンドで穴を開けたいなど。

## コマンドの構文
```
execute [ターゲットセレクター] [座標] [実行コマンド]
```
