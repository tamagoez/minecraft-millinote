# /tag
プレイヤーやエンティティにタグを付与/除去し、ターゲットセレクターで識別することができます。  
ツイートをエンティティとして考えると、ツイートをハッシュタグで絞るような感じです。

## 使用例
特定のプレイヤーにタグを付与し、タグを所持しているプレイヤーのみをテレポートする。

### Java Edition
```
nameをtargetに追加する
tag <targets> add <name>

targetに付与されているtagを表示する
tag <targets> list

nameをtargetから除去する
tag <targets> remove <name>
```

### Bedrock Edition
#### nameをtargetに追加する
```
tag [ターゲットセレクター] add <タグ名>
```
#### targetに付与されているtagを表示する
```
tag [ターゲットセレクター] list
```
#### nameをtargetから除去する
```
tag [ターゲットセレクター] remove <タグ名>
```

## ターゲットセレクター
**ターゲットセレクター**で絞り込みことができます。  
```
@<絞り込み>[tag=<タグ名>]
```
例: **admin**というタグのついた全プレイヤーを指定する  
`@a[tag=admin]`
