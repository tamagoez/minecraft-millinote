# コマンド
コマンドはチートの一種であり、コマンドを使うことで**RPG**や**戦闘ゲーム**などもつくることができます。  
詳しくは、[YouTube](https://youtube.com)などで「マイクラ バトロワ」などと検索すると例を見つけられると思います。

コマンド一覧へジャンプするには[ここから](#コマンド一覧)

## 必要設定
### Java Edition
**チートを許可する**必要があります。  
OP権限が必要なコマンドがあります。
### Bedrock Edition
チートの実行を許可する必要があるコマンドがあります。

<details>
    <summary>なぜ英語版も乗っているのか</summary>
    <div>
    英語版のWikiがある理由は、英語版のほうが圧倒的に**記事の数が多く**、**コンテンツの内容が濃い**ためです。
    #### 例え
    あなたは滑らかなクオーツブロック(Bedrock版)のデータ値を調べようとしています。  
    このような時、普通はGGって日本語版の記事に行き着くと思いますが、**今のところ日本語版には掲載されていません**。  
    じゃあどうすればいいのかというと英語版の記事を探してみてください。  
    **英語版には掲載されています**。  
    このようなことは特に**コマンドで起きます**。  
    なのでWikiNoteでは、英語版のWikiのURLも掲載しています。
    </div>
</details>
