# /execute
プレイヤーやエンティティとしてコマンドを実行させることができます。

## 使用用途のイメージ
特定のプレイヤーまたはエンティティの相対座標を使用してコマンドを実行したいときなど。

## コマンドの構文
**Java版とBE版でコマンドの構文が全く異なります**。
追記: **今後のアップデートでBE版の構文が廃止され、Java版の構文と統合される可能性があります**

### Java Edition
```
execute ...
```

### Bedrock Edition
```
execute [ターゲットセレクター] [座標] [実行コマンド]
```
座標は相対座標でも可能です。  
実行コマンドのセレクターは、**@s** または **@p**で自己を指定できます。

## コマンド例
### 全プレイヤーの下を1ブロック掘る
#### fill
```
execute @a ~~~ fill ~ ~-1 ~ ~ ~-1 ~ air
```
#### setblock
```
execute @a ~~~ setblock ~ ~-1 ~ air
```
