# testfor
**レッドストーンコンパレータ**や**条件付きコマンドブロック**と連結させることで、対象となるターゲットセレクターが存在しているか検知することができます。

対象となるターゲットセレクターとしてコマンドを実行したいなら、/executeコマンドを参考にしてください。

## 使用例
誰かがメインハンドにエンダーアイを持ったらPVPゲームを開始する。

## コマンドの構文
**常時検知するにはリピーター(反復)を使用する必要があります。**  
**レッドストーンコンパレータ**または**条件付きコマンドブロック**を接続しないと検知できません。
### Java Edition
### Bedrock Edition
```
testfor [ターゲットセレクター]
```

## コマンド例
### Java Edition
### Bedrock Edition
#### コマンドブロックの周辺5メートル以内に人がいたらキルする
```
testfor @a[m=5]
```
**レッドストーンコンパレータ**
```
kill @a[m=5]
```
