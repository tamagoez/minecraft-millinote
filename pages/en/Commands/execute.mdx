# /execute
Execute command as player or entity.

## For example...
Use `/fill` command with absolute path as each player.

## Command Syntax
### Java Edition
```
execute ...
```

### Bedrock Edition
```
execute [target selector] [coordinate] [command]
```
You can use **relative** coordinate.  
Relative coordinate set up each target selector's coordinate the standard coordinate.
